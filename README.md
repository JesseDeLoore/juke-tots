# Juke-Tots

A DIY maker project to build a digital record player that allows anyone to easily play a set of music files

## 🗂️ Repository Layout

- **BOMs/** – parts lists & cost breakdowns  
- **design/** – CAD exports, PCB schematics, enclosure files  
- **firmware/** – Python scripts, tag logic, audio player code  
- **docs/** – assembly guide, user manual, troubleshooting  
- **scripts/** – utility scripts (e.g. dump tag UIDs)
